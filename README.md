# Unit---5-CSV-JSON-file-using-python
The Library Management System includes several important features that help in managing library operations efficiently. It allows users to add new books by entering a unique book ID and book name. The system provides an option to view all available books in the library, making it easy to check the collection. Users can search for a specific book using its name, which helps in quick identification.The system also supports borrowing of books, where a student can borrow a book if it is available, and the details are stored accordingly. It includes a return book feature that allows users to return borrowed books and updates the records. Additionally, the system provides an option to remove books from the library when they are no longer needed.A menu-driven interface is used to make the system user-friendly and easy to navigate. The application uses dictionaries to store and manage data efficiently. It also performs validation checks to ensure correct input and proper functioning of operations.
